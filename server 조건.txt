[개인적으로 진행]
1. 지정 멤버만 접속 가능 -> loT반 멤버(김중규, 정재현, 허민재, 이윤서, 윤하얀, 전준용, 윤석현, 박희창, 진정현, 김지욱, 성재민, 강병헌, 김자연, 김영기)
2. 관리자 이름이 따로 있다 -> admin
3. 관리자 권한이 있다.
	3-1. 강제 퇴장
	3-2. 전체 공지
	3-3. 전체 채팅 비활성화
	3-4. 강제 퇴장 해제
4. 유저간 귓속말 가능
	command: /w 대상 할 말 
	ex) /w 이윤서 안녕하세요
5. 귓속말 받은 사람 출력 형태
	: 보낸사람 >>> 텍스트 형태로 옴 
	ex) 이윤서 >>> 안녕하세요
6. 500ms 이내에 채팅 반복 금지
7. 비속어 필터 -> 외부 파일을 통해서 필터 (데이터 베이스나 텍스트 파일을 이용)

8. 귓속말 유저명 or IP 차단
	/차단 유저명이나 ip하면 해당 ip나 유저에게 오는 귓말 차단
	/차단해제 유저명 or ip 차단해제
	ex) /차단 김지욱
	ex) /차단 192.168.31.189
	-> 김지욱이 귓속말을 보내면 김지욱 클라이언트 창에 " 차단당했습니다. " 출력
